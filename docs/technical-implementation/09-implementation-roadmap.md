# Technical Implementation Roadmap

## Phase 1: Application Foundation

Deliver:

- Angular app scaffold.
- Spring Boot app scaffold.
- Public status endpoint.
- Public category metadata endpoint.
- Local PostgreSQL Compose configuration.

Status: in progress.

## Phase 2: Domain Model and Migrations

Deliver:

- User profile model.
- Category model.
- Business model.
- Deal model.
- Voucher model.
- Redemption model.
- Flyway migrations.

## Phase 3: Business and Deal Workflows

Deliver:

- Merchant business creation.
- Admin business approval.
- Merchant deal creation.
- Admin deal approval.
- Public deal listing.

## Phase 4: Voucher Claiming

Deliver:

- Customer claim endpoint.
- Unique voucher code.
- QR token generation and hashing.
- Voucher wallet UI.

## Phase 5: Voucher Redemption

Deliver:

- Merchant redemption page.
- Manual code redemption.
- QR scan redemption.
- Transactional single-use enforcement.
- Redemption history.

## Phase 6: Security, Analytics, and Polish

Deliver:

- Identity provider integration.
- Role and membership authorization.
- Merchant analytics.
- Admin metrics.
- End-to-end tests.
- Deployment documentation.
