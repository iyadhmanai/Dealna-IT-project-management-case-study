# RACI Matrix

## RACI Definitions

- Responsible: Does the work.
- Accountable: Owns the outcome and final decision.
- Consulted: Gives input before decisions or delivery.
- Informed: Receives updates.

## Project Roles

| Role | Description |
| --- | --- |
| PM | Project Manager |
| PO | Product Owner |
| DEV | Developer |
| QA | Tester or quality owner |
| ADM | Admin or operations representative |
| MER | Merchant representative |
| CUS | Customer representative |
| SP | Sponsor or business owner |
